2. MATHEMATICAL NOTATION RENDERING
================================================================

This course contains heavy probability notation (summations, integrals, 
fractions, Greek letters, conditional probability bars, binomial 
coefficients, exponents). Use KaTeX to render all math.

REQUIRED IMPLEMENTATION:

Add to the <head> of every HTML page:

  <link rel="stylesheet" 
        href="https://cdn.jsdelivr.net/npm/katex@0.16.11/dist/katex.min.css">
  <script defer 
          src="https://cdn.jsdelivr.net/npm/katex@0.16.11/dist/katex.min.js"></script>
  <script defer 
          src="https://cdn.jsdelivr.net/npm/katex@0.16.11/dist/contrib/auto-render.min.js"></script>

Add to the end of <body>:

  <script>
    document.addEventListener("DOMContentLoaded", function() {
      renderMathInElement(document.body, {
        delimiters: [
          {left: "$$", right: "$$", display: true},
          {left: "$", right: "$", display: false}
        ],
        throwOnError: false
      });
    });
  </script>

Math is written inline as $...$ and display as $$...$$.

NOTATION CONVENTIONS USED IN THIS COURSE:

  Summation:               \sum_{i=1}^{n}
  Integral:                \int_{a}^{b}
  Fraction:                \frac{a}{b}
  Square root:             \sqrt{x}
  Intersection:            \cap
  Union:                   \cup
  Conditional probability: P(A \mid B)
  Complement:              A^c
  Greek letters:           \mu, \sigma, \rho, \Omega, \epsilon, \pi
  Binomial coefficient:    \binom{n}{k}
  Approximately equal:     \approx
  Infinity:                \infty
  Exponential:             e^{-x^2/2}
  Greater/less than equal: \geq, \leq
  Not equal:               \neq

WHEN INGESTING THE DOCX:

The course text is currently written in plain-text mathematical notation 
(e.g., "P(A and B) / P(B)", "sigma", "sqrt(x)", "x^2"). Convert these to 
proper LaTeX delimited by $...$ during ingestion. Specifically:

  "P(A and B)"     -> $P(A \cap B)$
  "P(A or B)"      -> $P(A \cup B)$
  "P(A | B)"       -> $P(A \mid B)$
  "sigma"          -> $\sigma$
  "mu"             -> $\mu$
  "sqrt(x)"        -> $\sqrt{x}$
  "x^2"            -> $x^2$
  "C(n, k)"        -> $\binom{n}{k}$
  "sum over..."    -> $\sum$
  "integral..."    -> $\int$
  "<="             -> $\leq$
  ">="             -> $\geq$
  "infinity"       -> $\infty$
  "approaches"     -> $\to$
  "epsilon"        -> $\epsilon$
